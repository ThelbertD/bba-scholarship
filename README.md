# BBA Scholarship Application

Static landing page for the Better Body Academy scholarship.

## Structure

- `index.html` — main scholarship application page
- `thankyou-jase/`, `thankyou-carly/`, `thankyou-dom/`, `thankyou-erin/` — coach-specific thank-you pages
- `img/` — page images
- `*.mp4` — member story videos
- The form on the main page routes to the correct thank-you page based on the coach the applicant selects.

## Local preview

```bash
npx serve -l 3000 .
```

Then open http://localhost:3000.
